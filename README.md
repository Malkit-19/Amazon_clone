# Amazon-Clone
A responsive Amazon India homepage clone built using HTML5 and CSS3. The project recreates Amazon's modern UI including navigation bars, hero section, product cards, promotional banners, and footer with responsive design.

# Amazon Clone

A responsive clone of the Amazon India homepage built using HTML5 and CSS3.

## Features

- Responsive Navigation Bar
- Search Bar
- Country & Language Selector
- Hero Banner
- Product Cards
- Promotional Sections
- Deals Section
- Recommendation Section
- Complete Amazon Footer
- Hover Animations
- Sticky Navbar

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
